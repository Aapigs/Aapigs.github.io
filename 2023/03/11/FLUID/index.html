Complete Tutorial: Hexo + Fluid Theme + GitHub Pages Deployment

Before getting started, you’ll need to register a free account on GitHub, which we’ll use later to deploy our static website. After creating your account, create a new repository with the name <your-username>.github.io. For example, if your username is demo, the repository name should be demo.github.io. You can leave the description blank, select Public so everyone can access your repository, and you don’t need to check the README option. Click Create repository to finish. You can find detailed steps online if needed!

1. Install Git

Git is a version control tool and the foundation for uploading your blog.

  • Windows:
    Go to https://git-scm.com/downloads, download and install using default settings (just keep clicking “Next”)

  • macOS:
    Run git --version in Terminal. If it’s not installed, the system will prompt you to install Xcode Command Line Tools - just follow the prompts

  • Linux:
    Use your package manager, for example: sudo apt install git

6. Configure the Fluid Theme

The Fluid theme’s configuration file is located at themes/fluid/_config.yml. Copy it to your root directory and rename it to _config.fluid.yml. This gives you higher priority when modifying configuration files. You can start with the default settings and adjust them later.
