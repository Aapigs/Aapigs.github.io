四则运算

**前言：**主要是记录一些基础的数学，最近发现不爱动脑筋，脑子都有一些不好了，逻辑数学都很混乱….

四则运算主要是常见的加减法运算，那么现在我们来回顾一下小数基础知识，他们分别的定义如下：

  • 加法：就是将两个或多个数合并成一个数的运算。例如，3 + 5 = 8，表示把3和5这两个数合并起来，结果是8。在实际生活中，加法可以用来计算总和。比如，小明有3个苹果，小红给他2个苹果，那么小明现在有3 + 2 = 5个苹果。
  • 减法：就是已知两个数的和与其中一个加数，求另一个加数的运算。例如，8 - 3 = 5，表示已知8是两个数的和，其中一个加数是3，那么另一个加数就是5。减法可以用来比较数量的多少。比如，小华有8本书，小强有3本书，小华比小强多8 - 3 = 5本书。
  • 乘法：就是求几个相同加数和的简便运算。例如，4 × 3 = 12，表示3个4相加，即4 + 4 + 4 = 12。乘法在计算面积、体积等方面很有用。比如，一个长方形的长是4米，宽是3米，那么它的面积就是4 × 3 = 12平方米。
  • 除法：就是已知两个因数的积与其中一个因数，求另一个因数的运算。例如，12 ÷ 3 = 4，表示已知12是两个因数的积，其中一个因数是3，那么另一个因数就是4。除法可以用来平均分配。比如，有12个糖果，要平均分给3个小朋友，每个小朋友可以得到12 ÷ 3 = 4个糖果。
