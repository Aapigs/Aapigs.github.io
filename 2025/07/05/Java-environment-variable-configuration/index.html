✅ Full Guide: Installing Java JDK and Configuring Environment Variables on Windows 11

🔧 Step 1: Download and Install Java JDK

  1. Visit the official Java website:

    • Oracle JDK: https://www.oracle.com/java/technologies/javase-downloads.html

    • Or use OpenJDK: https://jdk.java.net

  2. Download the installer and run it. During installation:

    • Click Next to accept the default settings.

⚙️ Step 2: Configure Environment Variables

✅ 2.1 Set JAVA_HOME

  1. Press Windows + S and search for “Environment Variables”.
    Click on Edit the system environment variables → then click Environment Variables.

  2. Under System variables, click New…
